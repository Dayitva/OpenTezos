### Question 8

```js
parameter (or (pair %assign string nat) (string %remove)) ;

       CDR ;

          { UNPAIR ; DIP { SOME } }
          { XXXXXXX };
       UPDATE ;
       NIL operation ;
       PAIR
     }
```

- [ ] DIP { NONE }
- [ ] DIP { NONE null }
- [x] DIP { NONE nat }
